# Heleeos - Blog

博客分为4个模块. 分别为 `数据`, `服务`,`展示`. **[线上个人博客](https://heleeos.com)**

## 数据

对应的项目为 `Blog-bean`.

使用的技术:Mysql, Redis, Mybatis

主要的功能有:
- 定义Bean
- 数据库读写
- Redis读写

## 服务

对应的项目为`Blog-manager`, `Blog-web`

使用的技术:Spring MVC

主要的功能有:
- 管理Blog
- 提供数据
- 提供API

## 展示

对应的项目为`Blog-web-NodeJS`

使用的技术:NodeJS

主要的功能有:
- 解析Markdown
- 生成静态文件
- 处理显示逻辑
